    class Player

    The Player class represents an object that contains all the basic information of a registered player. It is managed during the course of any game.

    Attributes

    curr_room[RW]

    The room's id the player is currently in.

    id[R]

    The player's id (Generated by the database engine).

    monster_tally[RW]

    The number of enemies the player has defeated.

    name[R]

    The player's username.

    strength[RW]

    The player's current strength.

    wealth[RW]

    The player's current wealth.

def initialize(options={:id=>0, :name=>"P", :strength=>50, :wealth=>50,                            :monster_tally=>0, :current_room=>nil})
  @id = options[:id]
  @name = options[:name]
  @strength = options[:strength]
  @wealth = options[:wealth]
  @monster_tally = options[:monster_tally]
  @curr_room = options[:current_room]
end
